2016 Florida Tax Returns
Comprehensive Analysis
Completed Project 2019

This is the tax returns for the State of Florida from 2016 provided by the US Treasury.
This data set is analyized using Python and the many libraries available within it
to determine many different aspects regarding tax brackets and information.

Download:

Data Set Python Code Formatted HTML Data Set PowerPoint

In This Project, The Following Questions were Answered Regarding Tax Data for Florida:

  • Where where the bulk of individuals living for the lower, middle and upper income brackets?
  • Which cities had the most retirement returns and brought in the most retirement income?
  • Which cities paid the most taxes?
  • Which bracket is paying the most in taxes?
  • What is the ratio of total income to taxable income across each bracket?
  • Which variables are statistically significant when it comes to taxable income?
