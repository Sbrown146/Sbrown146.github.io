  • How To Read A Binary Clock:

    LEDs represents 2 to some power with the first being 20 (1), the second 21 (2), the third 22 (4) and the last 23 (8).

    0 + 0 = 0
    Minutes

    For minutes, no LEDs are on which is 0.

    For seconds, no LEDs are on which is 0.

    Combining these two together will yield 0 + 0 = 0 minutes.
